# CosmoBrain — AI Search for Universal-Scale Information Networks

A high-fidelity, immersive single-page website for the **CosmoBrain** initiative
(FloLabs Group / Texas Advanced Robotics Research Lab — TARRL).

An open, remote research and product initiative using AI to study cosmic-scale
structure, signals, anomalies, and information-flow patterns across public space data.

## Live preview

Open `index.html` in any modern browser — no build step, no dependencies to install.

If you publish with **GitHub Pages**, set the source to the `main` branch / root
and the site will be served at `https://<your-username>.github.io/<repo-name>/`.

## Project structure

```
.
├── index.html        # All page content & section markup
├── css/
│   └── cosmos.css    # Cosmos design system: dark space theme, glassmorphism, motion
└── js/
    └── site.js       # Starfield + cosmic-web canvases, loader, scroll reveals,
                       # text-scramble headlines, rail/menu navigation
```

## Design

- **Concept:** cosmos — futuristic, atmospheric, immersive, editorial.
- **Visuals:** dark space backgrounds, glowing gradients, orbital lines, star
  particles, soft blur, glassmorphism, floating cards, layered depth.
- **Interaction:** smooth scrolling, animated section reveals, hover states,
  subtle parallax, floating elements, text-scramble headlines, a live section
  rail and an index overlay.
- **Type:** Space Grotesk (display/body) + Space Mono (labels), loaded from Google Fonts.
- **Responsive:** adapts across desktop, tablet, and mobile.
